# patric-tests

To run login test :

    git clone https://github.com/jaredbischof/patric-tests.git
    cd patric-tests/
    ./scripts/login.py --firebug [username] [password]

To run workspace test :

    git clone https://github.com/jaredbischof/patric-tests.git
    cd patric-tests/
    ./scripts/test.py --firebug [username] [password]
